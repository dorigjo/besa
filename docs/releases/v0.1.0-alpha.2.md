# v0.1.0-alpha.2 — npm alpha package release

Besa is now publicly installable as an npm alpha package.

Highlights:

* Published package: @dorigjo/besa@alpha
* CLI executable works through npm exec
* Package metadata is aligned with the public GitHub repository
* Manifest signing and verification are working
* Admission decisions are working
* Grant-aware admission is working
* Signed receipts are working
* Tests, smoke test, and npm pack dry-run pass

Install:

```
npm install @dorigjo/besa@alpha
```

Run:

```
npm exec --yes --package="@dorigjo/besa@alpha" -- besa --help
```

Status:
Alpha. Not production-stable yet.
