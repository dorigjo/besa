Agent Action Receipts

Besa

Signed receipts for AI-agent tool calls.

Before an AI agent touches a real tool, Besa verifies the manifest, checks the grant, decides admission, and emits a signed receipt. 

npm install @dorigjo/besa@alpha

The problem

AI agents are getting permission to call real systems: CRMs, databases, payment systems, deployment APIs. Most teams cannot prove what was allowed, denied, or executed.

The wedge

Besa signs manifests, checks grants, controls admission, and produces verifiable receipts.

The primitive

Besa is the receipt layer for AI-agent actions.

CLI

npm exec --yes --package="@dorigjo/besa@alpha" -- besa --help

besa keys
besa sign examples/manifest.yaml
besa verify examples/manifest.signed.json
besa admit examples/manifest.signed.json crm.lookup
besa admit examples/manifest.signed.json crm.delete
besa receipt crm.lookup examples/manifest.signed.json

Status

Public alpha. Local-first. No hosted backend yet. Not production-stable.
